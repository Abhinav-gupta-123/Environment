## 🚀 Quick Start

1. **Clone this repo**:
   ```bash
   git clone https://github.com/<your-username>/<repo-name>.git
   cd <repo-name>
````

2. **Create and activate a virtual environment**:

   * **Windows**:

     ```bash
     python -m venv .venv
     .venv\Scripts\activate
     ```
   * **Linux/macOS**:

     ```bash
     python3 -m venv .venv
     source .venv/bin/activate
     ```

3. **Install all packages**:

   ```bash
   pip install -r requirements.txt
   ```
